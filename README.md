
# Private-GPT(This is just for fun project)

A private, local implementation of GPT for secure document processing and querying.

## Features

- 🔒 **Privacy-First**: Run entirely locally without sending data to external services (ollama)
- 🚀 **Fast**: Optimized for quick inference and response times
- 📄 **Document Support**: Process multiple document formats
- 🔍 **Semantic Search**: Query documents using natural language

## Technologies Used
- FastAPI for the backend API
- Ollama for local LLM inference
- ChromaDB for vector storage and retrieval
- LangChain for document processing and LLM interaction
- HTML/CSS/JavaScript for the frontend interface(Dont ask about the frontend, its by AI)
